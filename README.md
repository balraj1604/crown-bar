# Crown Bar — Mega Max 40000

Interactive flavour-slider landing page for the Al Fakher Crown Bar Mega Max 40000 disposable.

- 20 flavours, each its own colour-world
- Real device cut from official renders, recoloured per flavour (branding preserved)
- GSAP smart-animate transitions: device pops up from the bottom, ingredients drop from the top, giant word slides right→left, background colour morphs (ease-in-out-back, ~2s)
- Single self-contained `index.html` + `assets/` device images

Open `index.html` (or serve the folder) to view.
